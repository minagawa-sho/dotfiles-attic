# dotfiles-attic

## Usage

```
// Run the install script
$ curl -LsS 'https://raw.githubusercontent.com/minagoro/dotfiles-attic/master/bin/install.sh' | bash -

// Repeat on root
$ su -
# curl -LsS 'https://raw.githubusercontent.com/minagoro/dotfiles-attic/master/bin/install.sh' | bash -

// All installation is done !
```
